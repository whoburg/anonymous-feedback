FAQ


Why should feedback be anonymous, anyway?

That's a great question. The purpose of this site is simply to enable true anonymity, when it is desired by a team.


Why do I need to install special software?

To jumble up the order of feedback responses and keep them anonymous, we need to store them on a server. We don't want anyone, including site admins, to be able to read this feedback, so we store it encrypted. To read it, you decrypt it locally on your computer using your secret private key. This ensures you are the only person who can read your personal feedback.


How is my data kept safe?

Feedback text that you author is sent to the server via a Secure HTTPS Socket, where it is encrypted using Public-key_cryptography. It is stored in this encrypted form until results are published. The only person who can decrypt it is the recipient, using their private key.
